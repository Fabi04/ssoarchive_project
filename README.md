# SSO Archive Project

An Angular application created to archive and manage SSO data. This project was built using Angular CLI version 21.1.4.

## Quick Start

### 1. Install Dependencies
Before running the application, make sure you install the required packages:
```bash
npm install
```

### 2. Run the Application
To start the local development server, run:
```bash
ng serve
```
Once the server is running, open your browser and navigate to `http://localhost:4200/`.

## Key Commands

* **Build the project:** `ng build` (Output files will be saved in the `dist/` directory)
* **Run tests:** `ng test`
